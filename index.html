Syarat & Ketentuan Pengumpulan

Mohon perhatikan 4 poin kunci ini agar tugas Anda sukses diunggah dan dinilai:

  • Ekstensi Aman pastikan file Anda berformat: PDF, DOC, DOCX, ZIP, RAR, atau 7Z.
  • Batas Ukuran file Anda maksimal 12 MB.
  • Format Nama KRITIS Wajib ikuti aturan penamaan ini:
    [Nama Lengkap]_[NIM]_[Mata Kuliah]_[Nomor Pertemuan]
    (Contoh: Putra_1955201011_PBO_P1)
  • Awas! Nama file yang salah format berisiko tinggi membuat tugas gagal terekam di sistem penilaian.
